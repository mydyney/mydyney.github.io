## 💡 中文翻译原则

### 1. 地名翻译
- **专有地名**: 使用标准中文译名（如：涩谷、银座、浅草）
- **新建筑**: 直译 + 保留原意（如：麻布台ヒルズ → 麻布台之丘）
- **品牌名**: 保持原音译或官方中文名（如：アンダーズ → 安达仕）

### 2. 美食翻译
- **日式特色**: 保留日语（如：居酒屋、寿司、拉面）
- **通用词汇**: 使用中文（如：グルメ → 美食、カフェ → 咖啡厅）
- **甜品类**: 本地化翻译（如：スイーツ → 甜品）

### 3. 旅游术语
- **攻略优先**: "ガイド" → "攻略" 而非 "指南"（更符合中国游客习惯）
- **实用性**: "情報" → "信息" 或 "攻略"
- **观光**: "観光" → "观光" 或 "景点"

### 4. 购物类
- **音译为主**: Don Quijote → 唐吉诃德
- **意译为辅**: アウトレット → 奥特莱斯
- **混合使用**: KALDI → KALDI咖啡（品牌+类别）

### 5. 活动类
- **节日**: 使用中文节日名（圣诞节、万圣节）
- **灯光秀**: イルミネーション → 灯光秀（而非 "彩灯"）
- **集市**: マーケット → 集市（而非 "市场"）

## 🎯 博客翻译指导原则

### 面向中国游客的表达方式

1. **标题优化**
   - ❌ 日式: 「〇〇の完全ガイド」
   - ✅ 中式: 「〇〇完全攻略」「必看！〇〇旅游指南」

2. **内容表达**
   - 强调性价比、实用性
   - 突出「必打卡」「网红」「ins风」等热门词汇
   - 使用「强推」「血泪经验」「避坑指南」等吸睛表达

3. **实用信息**
   - 营业时间用「开放时间」而非「营业时间」
   - 价格信息加「约」「人民币」换算
   - 交通信息详细（中国游客最关心）

4. **文化适配**
   - 删除日本本地人视角内容（如免税相关）
   - 增加中国游客关心的点（如会说中文、支持支付宝/微信支付）
   - 调整文化参考（用中国节日类比日本活动）

5. **语气风格**
   - 热情、直接、带点网络用语
   - 多用感叹号、emoji（适度）
   - 「超级推荐！」「绝对不能错过！」
